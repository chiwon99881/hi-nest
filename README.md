## Learning Nest

- #01 Init

  ```bash
  npm install -g @nestjs/cli

  nest new (create nest project)
  ```

- #02 Create New Controller, Service

  ```bash
  nest generate controller

  nest generate service
  ```

- #03 DTO with Validation

  ```bash
  npm i class-validator

  npm i class-transformer
  ```
